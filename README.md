# Customer Behavior Statistical Analysis

Analyzes customer spending, churn, and campaign performance using inferential statistics.

## Questions Answered
1. Descriptive stats (mean, median, mode)
2. Outlier detection (IQR)
3. Skewness & kurtosis
4. Gender spending difference (t-test)
5. Product category vs. churn (chi-square)
6. Regional spending (ANOVA)
7. Campaign performance (t-test)
8. Normality check (Shapiro-Wilk + Q-Q plot)
9. Central Limit Theorem demonstration
10. 95% confidence interval for mean spend
